# 🐦 Flappy Bird Clone (Pygame)

This is my custom version of the classic **Flappy Bird** game, developed using **Python** and **Pygame**.  
It features custom-designed assets, PNG-based score display, smooth controls, and a persistent high score system.

---

## 🎮 Features

- 🖼️ Custom assets with organized folders (`pngs/`, `musics/`)  
- 🎵 Sound effects for flapping, scoring, and collisions  
- 🔢 PNG-based score rendering  
- 📈 High score saved to a text file (`skor.txt`)  
- 🚀 Smooth 60 FPS gameplay  
- 🧠 Clean and beginner-friendly codebase  

---

## 🖥️ How to Run

1. Make sure Python is installed on your system.  
2. Install Pygame by running:

   `pip install pygame`

3. Launch the game by running:

   `python flappy_bird.py`

---

## ⌨️ Controls

- Press `SPACE` to flap  
- Press `SPACE` again after game over to restart  
- Close the window to quit the game
